# Dynamic Hero Banner Deployment Package

This package contains only the `dynamicHeroBanner` Lightning Web Component.

## Deploy From Directory

```sh
sf project deploy start --metadata-dir deployable-packages/dynamicHeroBanner
```

## Deploy From ZIP

```sh
sf project deploy start --metadata-dir deployable-packages/dynamicHeroBanner.zip --single-package
```

## Contents

- `package.xml`
- `lwc/dynamicHeroBanner/dynamicHeroBanner.html`
- `lwc/dynamicHeroBanner/dynamicHeroBanner.js`
- `lwc/dynamicHeroBanner/dynamicHeroBanner.css`
- `lwc/dynamicHeroBanner/dynamicHeroBanner.js-meta.xml`
